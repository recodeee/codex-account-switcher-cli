# Plan Workspace: ralplan-openspec-plan-export

This folder is the durable plan workspace for this plan.

## Structure

- `summary.md` — full plan summary, ADR, acceptance criteria, and verification plan
- `planner/` — workflow/export contract work
- `architect/` — ownership boundaries and artifact model
- `critic/` — guardrails and idempotency review
- `executor/` — implementation work
- `writer/` — docs and command help updates
- `verifier/` — end-to-end validation and evidence

Each agent folder can hold notes, task breakdowns, checklists, and outputs for that role.
