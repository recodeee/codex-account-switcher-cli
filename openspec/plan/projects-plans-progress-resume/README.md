# Plan Workspace: projects-plans-progress-resume

This folder stores durable planning artifacts before implementation changes.

## Role folders
- `planner/`
- `architect/`
- `critic/`
- `executor/`
- `writer/`
- `verifier/`
- `designer/`

Each role folder contains `tasks.md` with visible Spec / Tests / Implementation / Checkpoints checklists.
Planner also gets `plan.md`; executor also gets `checkpoints.md`.
Planner plans should follow `openspec/plan/PLANS.md`.
